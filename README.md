# 电力短期负荷预测 | EnergyAI Project 1
基于LSTM实现电网短期24h负荷预测，面向智能电网、虚拟电厂、能源成本优化。

## 项目背景
双碳目标下，负荷预测是调度、用电优化、峰谷调节的核心基础能力。

## 技术栈
- Python 数据分析
- Pandas / NumPy 数据处理
- Matplotlib 可视化
- Scikit-learn 机器学习
- TensorFlow / PyTorch 深度学习
- LSTM 时间序列预测模型
